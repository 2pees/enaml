destroy()[source]¶

A reimplemented destructor.

Qt takes ownership of the menubar, so the destructor does not attempt to unparent the menubar. The child_removed handler on the main window will reset the menu bar.

destroy()[source]¶

A reimplemented destructor.

This destructor simply drops the reference to the menu bar and the enaml declaration and clears the menus in the menu bar. Destroying it will cause wx to segfault.
